### Software

Tools used in your work will many time stray from your personal preferences but I see this as an advantage as it allows for distinct environments for distinct kinds of work. It also keeps things fresh and makes switching gears easier (at least for my brain).

* Work environment
  * IDE: JetBrains PhpStorm
  * Browser: Mozilla Firefox
  * Terminal: iTerm 2 (Oh My Zsh)
  * Server management: MAMP Pro
  * Database management: Sequel Ace

  * Design system management: Pattern Lab

  * IDE: Microsoft VSCode
  * Browser: Google Chrome
  * Terminal: WSL2 Arch (bash)
  * Environment management: Docker
  * KVM: input-leap
